Обновление ncrew.sh и документации

**Тип:** DevOps / Docs
**Приоритет:** Medium

**Описание:**
Обновить скрипт запуска и документацию, чтобы они соответствовали новой структуре.

**Задачи:**
1.  Обновить `ncrew.sh`:
    -   Проверка наличия Python.
    -   Создание venv.
    -   Установка зависимостей.
    -   Запуск `python main.py`.
2.  Обновить `README.md` (или `PROJECT_INDEX.md`):
    -   Описать новую структуру хранения данных (`~/.ncrew`).
    -   Инструкция: как создать первый проект через UI или CLI (если останется).
    -   Убрать упоминания `.env` для настройки ботов.

**Критерий приемки:**
- `bash ncrew.sh` успешно запускает приложение на чистой машине (при наличии зависимостей ОС).
- Документация актуальна.
