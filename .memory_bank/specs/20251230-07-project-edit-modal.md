# Спецификация: Project Edit Modal

**Имя:** Project Edit Modal  
**Дата:** 2025-12-30  
**Версия:** 1.0  
**Статус:** Done

## Обзор

Добавить возможность редактирования настроек уже добавленного проекта через модальное окно.

## User Stories

### US-6.1: Редактирование проекта
Как **Разработчик**, я хочу редактировать настройки проекта, чтобы изменить название, префикс или дефолтную модель.

## Требования

### Функциональные требования

#### 1. Редактируемые поля
- `name` - название проекта
- `worktreePrefix` - префикс для работы
- `defaultModel` - дефолтная модель

#### 2. Read-only поля
- `path` - путь к проекту (не редактируется)

#### 3. Предупреждение
- При изменении `worktreePrefix` и существовании worktree с другим префиксом:
  - Показать предупреждение
  - Старые worktree остаются

#### 4. Backend
- API endpoint: `PUT /api/projects/:id`
- Валидация модели
- Обновление конфига проекта

#### 5. Frontend
- Компонент `ProjectEditModal`
- Кнопка "Edit" на странице проекта
- Форма с полями для редактирования

### Нефункциональные требования

1. **Безопасность:** Поле `path` только для чтения
2. **Предупреждение:** Понятное сообщение о последствиях изменения worktreePrefix

## Приемочные критерии

1. ✅ На странице проекта есть кнопка "Edit"
2. ✅ При нажатии открывается модалка редактирования
3. ✅ Можно редактировать `name`, `worktreePrefix`, `defaultModel`
4. ✅ Поле `path` только для чтения
5. ✅ При изменении `worktreePrefix` показывается предупреждение
6. ✅ При сохранении настройки обновляются в конфиге проекта
7. ✅ API endpoint `PUT /api/projects/:id` работает
