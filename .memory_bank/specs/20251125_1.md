Редактор ролей и глобальных промптов

**Тип:** Feature / Fullstack
**Приоритет:** High
**Workflow:** [feature.md](../workflows/feature.md)

**Описание:**
Переработать страницу "Настройки ролей" для поддержки новой архитектуры (конфиг проекта + глобальные промпты).

**Задачи:**
1.  **Разделение ответственности:**
    -   Левая часть (или отдельная вкладка): Редактор `config.yaml` текущего проекта. Поля: `role_name`, `agent_type`, `cli_command`, `telegram_bot_token`, выбор файла промпта (dropdown).
    -   Правая часть (или модальное окно/вкладка): Редактор файлов в `~/.ncrew/prompts/`.
2.  **Редактор промптов:**
    -   Список файлов `.md` из глобальной папки.
    -   Возможность создать новый файл, редактировать текст, сохранить.
3.  **Связывание:**
    -   В настройках роли поле "Prompt File" должно быть выпадающим списком существующих файлов из папки промптов.

**Критерий приемки:**
- Пользователь может создать новый промпт "architect_v2.md".
- Пользователь может выбрать этот промпт для роли "Architect" в текущем проекте.
- Пользователь может вписать токен бота для роли в UI, и он сохранится в `config.yaml`.
