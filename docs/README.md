## Ключевые концепции

### Задачи (Tasks)
Задачи описываются в MD-файлах в папке `.memory_bank/tasks/` вашего проекта. Каждая задача содержит:
- Название
- Статус (New, In Progress, Done, Failed)
- Приоритет (Low, Medium, High)
- Описание задачи

### Git Worktree
Каждая задача выполняется в изолированном Git Worktree, чтобы изменения не влияли на основную ветку или другие задачи.

### Параллельное выполнение
Несколько задач могут выполняться одновременно, используя разные Worktree.
