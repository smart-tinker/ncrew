# ncrew

ncrew — это открытый терминальный интерфейс для совместного взаимодействия с CLI-клиентами популярных ИИ-агентов. Цель MVP — воспроизвести ключевые функции консольного группового чата по мотивам consilium-agent-tui, сохранив простоту и возможность свободной модификации.

## Основные возможности MVP
- **Групповой чат** между пользователем и несколькими агентами с последовательными ответами и общей историей сообщений.
- **Интеграция с внешними CLI**: Claude Code, Codex, Gemini CLI, Qwen CLI и Opencode запускаются как отдельные процессы.
- **Минимальные горячие клавиши** для отправки сообщений, вызова помощи и обновления конфигурации.
- **Конфигурация** через файл `~/.ncrew/settings.json` с описанием доступных агентов и параметров TUI.
- **Готовность к расширению**: структура проекта и документация упрощают подключение дополнительных клиентов и функций.

## Установка и запуск
1. Убедитесь, что на машине установлен Python 3.10+.
2. Установите зависимости проекта (режим разработки):
   ```bash
   pip install -e .
   ```
3. Скопируйте пример `examples/settings.sample.json` в `~/.ncrew/settings.json` и при необходимости отредактируйте команды либо воспользуйтесь шаблоном из [руководства пользователя](docs/user-guide.md).
4. Запустите интерфейс:
   ```bash
   ncrew
   ```
   или
   ```bash
   python -m ncrew
   ```
   При необходимости можно указать альтернативный файл настроек:
   ```bash
   ncrew --config /path/to/settings.json
   ```

## Тестирование

Для запуска автоматических проверок и тестов используйте `pytest`:

```bash
pip install -e .[dev]
pytest
```

## Дорожная карта MVP
1. Реализовать возможности историй 1–2: подготовка конфигурации, запуск TUI и отображение состояния (выполнено).
2. Доработать интеракции истории 3: обмен сообщениями и последовательные ответы агентов (выполнено на уровне вызова внешних CLI).
3. Закрыть историю 4: горячая перезагрузка настроек и сообщения статуса (выполнено).
4. Планировать расширения поверх согласованного сценария (история сообщений, дополнительные хоткеи, аналитика) в следующих итерациях.
